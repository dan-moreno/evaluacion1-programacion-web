INICIO EXPERIENCIA PROYECTOS CONTACTO

Proyectos

Actividad 1: Esta actividad consistió en elaborar nuestro primer sitio web, en el cual incluímos algunas de nuestras características como personas, así como habilidades y hobbies. Esto con el fin de practicar la sintaxis del lenguaje HTML.

Actividad 2: A través de esta actividad se puso en práctica el manejo de tablas en HTML, desde su creación hasta la introducción de datos en ellas, además de algunas nociones básicas para darles estilo (colores de fondo, bordes, etc).

Actividad 3: Se realizó un formulario simple utilizando HTML con el fin de estudiar las principales etiquetas que conforman este elemento. También se incluye un botón para guardar información y posteriormente darle funcionalidad.

Actividad 4: Se consolidaron en un mismo sitio todos los conceptos vistos con anterioridad, para crear un sitio que incluye no sólo nuestras características, sino también una tabla con nuestras referencias y un formulario de contacto.

Actividad 6: Se elaboró un nuevo formulario con nuevas etiquetas, para luego verificar la calidad del sitio web utilizando el validador en línea de la organización W3C. Así nos aseguramos de que el sitio cumple con los estandares.

Actividad 7: Luego de revisar los conceptos fundamentales de CSS, se elaboró un formulario simple y se agregaron una serie de estilos en base a una pauta específica. Gracias a ello fue posible visualizar el potencial de las hojas de estilo.

Actividad 8: Finalmente, se utilizó un sitio web educativo como base para un ejercicio de estilos. Mediante esta actividad se pudo poner a prueba la destreza y creatividad del desarrollador cuando se trata de perfeccionar la parte visual.
